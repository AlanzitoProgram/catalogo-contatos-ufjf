# Melhorias aplicadas e próximas ideias

## Aplicadas nesta versão

- Separação entre HTML, CSS, JavaScript e JSON.
- Busca instantânea.
- Filtros combinados.
- Favoritos com LocalStorage.
- Exportação CSV.
- Layout responsivo.
- Tema claro/escuro.
- Botões para copiar telefone e e-mail.
- Correção automática de URLs duplicadas.

- Criar painel administrativo simples para editar contatos sem mexer no JSON manualmente.
- Adicionar QR Code por contato.
- Criar páginas individuais por setor.
- Adicionar campo de observações visível.
- Adicionar validação automática de links quebrados.
- Adicionar versão PWA para funcionar melhor no celular.
- Integrar com Google Sheets como base de dados.
